# Vue3 + Tailwind pet-project webmap

First and simple web application to visualise remote sensing data from public WMS sources.
[Live demo](https://toshaklg.github.io/tw-showcase-webmap/)

## Useful commands

Install npm modules:
```bash
npm run i
```

Run locally:
```bash
npm run dev
```

Build:
```bash
npx run build
```
